# 🇮🇹 Italy Travel Project

Бұл жоба Италияның көрікті жерлеріне арналған ақпараттық веб-сайт. Жоба оқу барысындағы үй тапсырмасы аясында тестілеуден өтіп, Netlify платформасына жарияланды.

## 🚀 Жоба туралы
Сайт қолданушыларға Италияның танымал қалалары (Милан, Венеция, Амальфи) туралы мәлімет береді және кері байланыс орнатуға мүмкіндік береді.

## 🛠 Қолданылған технологиялар
* **HTML5** - сайттың құрылымы
* **CSS3** - дизайн және адаптивтілік (Responsive design)
* **JavaScript** - интерактивті элементтер және форма валидациясы
* **Git/GitHub** - нұсқауларды басқару
* **Netlify** - хостинг және деплой
